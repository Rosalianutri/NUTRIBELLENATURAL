✅ Qué hacer para que funcione

  1. Guarda este código en un archivo llamado index.html.

  2. Pon la imagen que me mandaste con el nombre JG.JPG en la misma carpeta donde está este index.html.

  3. Súbelo a un hosting gratuito, por ejemplo:

    • GitHub Pages (creas un repositorio nuevo, subes index.html y la imagen, activas Pages)

    • Netlify (arrastrar carpeta o conectar repositorio con tu cuenta gratuita)

    • Google Drive + servicio de hosting gratuito (menos recomendado)

  4. Accede a la URL que te da el servicio y ¡listo! tendrás tu página online.
